Project Overview

My Portfolio Website is a fully responsive and modern personal site built using HTML, CSS, and JavaScript. It is designed to highlight my professional identity, showcase the projects I've built, and provide easy access to my resume and contact details. The layout includes smooth scrolling, interactive navigation, section-based content (About Me, Skills, Projects, and Contact), and is optimized for both desktop and mobile devices. It features animated buttons, organized content blocks, and external links to platforms like GitHub and LinkedIn.

I developed this site to serve as a digital portfolio where recruiters, collaborators, or potential clients can get a quick, visually appealing summary of my skills and experience. It reflects not only my frontend skills but also my attention to user experience, UI structure, and semantic coding practices.
